
# Simiriki — Pretty Kit (v1)

**Cómo usar**
1) Sube todos los archivos a tu repositorio de Azure Static Web Apps.
2) Los enlaces de `Agenda` y del `iframe` ya han sido configurados con tus URLs de producción. Si cambias de formulario o agenda, actualiza las URLs en `index.html`.
3) Opcional: reemplaza `assets/img/logo.svg` (usado también como favicon) por tu logotipo final.
4) Ejecuta un Lighthouse audit y corrige cualquier alerta.

**Colores**
- Primario: #19C37D
- Neutros: escala Slate

**Tipografía**
- Inter (opcional) con fallbacks del sistema.

**Compatibilidad**
- Dark mode automático (prefers-color-scheme).
